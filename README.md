README File SDLC of Library Management System
This project is a simple Library Management System written in Python. It was developed as part of the SEN 201 course to help understand the basic concepts of software engineering and the Software Development Life Cycle (SDLC).
The system allows books to be added to a library and displays all available books.
Features
1. Add books to the library
2. Display all books in the library
3. Simple and beginner-friendly Python code
Tools Used
1. Python Programming Language
2. GitHub for version control and project submission
How to Run the Program
1. Install Python on your computer
2. Download or clone the project repository from GitHub
3. Open the project folder
4. Run the program using the command:
   python library_management_system.py
